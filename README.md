# Daily Wordle Variants

Each day is a new variant of wordle.

Access Here: https://lualum.github.io/WordleVariants/

## Weekly Game Schedule

The game type is determined by the current day of the week. Here is the rotation schedule:

| Day           | Game Mode    | Description                                                                         | Max Guesses |
| ------------- | ------------ | ----------------------------------------------------------------------------------- | ----------- |
| **Monday**    | **Fakele**   | One letter in the evaluation is a "fake" letter!                                    | 7           |
| **Tuesday**   | **Hardle**   | "Hard Mode"â€”any revealed hints must be used in subsequent guesses.                  | 6           |
| **Wednesday** | **Yellodle** | The goal is flipped: win by making all letters **yellow** (present but wrong spot). | 6           |
| **Thursday**  | **Gamble**   | One random letter in your result provides no information (marked as hidden).        | 7           |
| **Friday**    | **Oddsle**   | The board starts with two random, incorrect guesses already filled in for you.      | 7           |
| **Saturday**  | **Duodle**   | Solve two different words simultaneously on two boards.                             | 7           |
| **Sunday**    | **Fourle**   | A shorter challenge using only four-letter words.                                   | 6           |
